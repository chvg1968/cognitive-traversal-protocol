# Code of Conduct

This project welcomes practical, respectful collaboration.

## Expected Behavior

- Be direct and constructive.
- Critique ideas, not people.
- Assume contributors are trying to improve the project.
- Keep discussions focused on clarity, usefulness, and accuracy.
- Respect different levels of experience with LLMs and agentic tools.

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Discriminatory language
- Repeated off-topic arguments
- Publishing private or sensitive information
- Bad-faith contributions intended to derail the project

## Enforcement

Maintainers may edit, hide, or remove comments and contributions that violate this code of conduct. Repeated or severe violations may lead to blocking from the project.

If you report an issue, include context and links where possible.
